# InFocus Film School — Meta Ad Case Study

An interactive case-study site for a take-home Meta ad challenge (1080×1350 px) for InFocus Film School's Graphic + Digital Design program. Built with plain HTML, CSS, and JavaScript — no frameworks.

Sections: hero with a magnifying-glass zoom on the final ad, design rationale, an interactive ten-stage process timeline, a before/after concept-shift slider, a numbered hotspot breakdown of the final ad, about, and contact.

## Open

Open `index.html` in a browser. No build step.

## Add your images

See `assets/README.txt` for the exact filenames each section expects (`final-ad.png`, `rationale.png`, `process-01.jpg` … `process-10.jpg`). Until they're added, each slot shows a clean placeholder naming the file it needs.

## Edit copy

- Process timeline text → `script.js`, `PROCESS_STEPS` array.
- Final ad hotspot labels/positions → `script.js`, `HOTSPOTS` array.
- Everything else → directly in `index.html`.
